# High / Low Card Game
This is a coding test done for Automated Insights.

[Demo](https://cardgame.allureprojects.com/)

## Initiate project
- Run `yarn`

## Run project locally
- Run `yarn start`

## Assumptions
- If the same value card is drawn, it counts as a correct guess
